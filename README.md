# spojovacka

Minimální “hello world” pro ověření GitHub Pages.

## Zapnutí GitHub Pages

1. V GitHub repu otevři **Settings → Pages**
2. V **Build and deployment** nastav:
   - **Source**: *Deploy from a branch*
   - **Branch**: `main`
   - **Folder**: `/(root)`
3. Ulož a počkej, až se objeví URL.

Pak by se měla zobrazit stránka z `index.html`.
